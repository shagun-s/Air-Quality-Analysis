# Delhi-Air-Quality-Analysis
Delhi Air quality analysis & Dashboard
# Objective
Analyze air quality trends across Delhi locations and identify the impact of weather conditions on AQI.

# Tools Used

-Python
-Pandas
-NumPy
-Matplotlib
-Seaborn
-Power BI

# Workflow

-Data Cleaning
-Outlier Treatment
-Feature Engineering
-Exploratory Data Analysis
-Dashboard Creation 

# Key Insights
-AQI peaks in May.
-Monsoon months show lowest pollution.
-Humidity negatively correlates with AQI.
-Dwarka & IGI Airport show higher pollution levels.
